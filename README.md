# 应急救援装备管理系统

## 介绍

一个基于SpringBoot，Vue，Element的应急救援装备管理系统，支持

## 快速开始

### source /Emergency_Supplies_Management_System/sql_0504.sql;

### mvn -X clean install
###  mvn package
### java -jar SpringBoot-0.0.1-SNAPSHOT.jar

### cd /Emergency_Supplies_Management_System/vue
### npm install
### npm run serve
